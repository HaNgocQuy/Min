# Min — Bút diện tử

Min là một dự án mô phỏng / điều khiển "bút điện tử" viết bằng Arduino. Mục tiêu chính của dự án là thu thập tín hiệu từ cảm biến, xử lý và mô phỏng hành vi của bút điện tử trong phần cứng/ứng dụng đơn giản.

Tệp chính:
- But.ino — sketch Arduino chính
- LICENSE — giấy phép dự án

Phần mềm & thư viện:
- Phần mềm:
  - Arduino IDE: https://www.arduino.cc/en/software
- Thư viện:
  - ElectronicCats - mpu6050 : https://github.com/electroniccats/mpu6050 
  - T-vK - ESP32-BLE-Mouse : https://github.com/T-vK/ESP32-BLE-Mouse
